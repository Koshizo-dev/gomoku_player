# gomoku_player


> Build:

`cargo build --release`

__Test mode__

> Quick run:

`cargo run --release -- --test <AI_PATH>`

> Or

`./target/release/gomoku_player --test <AI_PATH>`

__Fight mode__

> Quick run:

`cargo run --release -- --fight <AI1_PATH <AI2_PATH>`

> Or

`./target/release/gomoku_player --fight <AI1_PATH> <AI2_PATH>`
